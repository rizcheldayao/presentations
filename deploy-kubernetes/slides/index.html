    • What is Kubernetes?

    Kubernetes is an open-source system for automating deployment, scaling, and management of containerized applications.

    - https://kubernetes.io/

    First, let's look into containers...

    What are containers?

  • Made up of container images
  • Consists of an executable package of an application (libraries, dependencies, etc)
  • Virtualize at the operating system level, with multiple containers running atop the OS kernel directly

    • https://cloud.google.com/containers/

    Benefits:

  • Lightweight & performant
  • Platform independent
  • Isolation & resource sharing
  • Ideal for microsservice architectures

    • How do we package our application to a container?

    That's where Docker comes into play...

    Docker is an open platform for developers and sysadmins to build, ship, and run distributed applications, whether on laptops, data center VMs, or the cloud.
