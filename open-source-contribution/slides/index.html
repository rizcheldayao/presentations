    What is open source?

    denoting software for which the original source code is made freely available and may be redistributed and modified.

    - Dictionary.com

    That means that anyone...

    • Can take the source code
    • Can modify it
    • Can distribute their own versions

      • There's open source technology everywhere

      • Operating systems
      • Programming languages
      • IoT
      • AI and many more...

        • Linux, Mozilla Firefox, MongoDB, Node.js

        Imagine a world where all technology was copyrighted and required licenses to use...

        How would it affect collaboration and innovation today?

        78 percent of respondents said their companies run part or all of its operations on OSS and 66 percent said their company creates software for customers built on open source.

        - 2015 survey conducted by Black Duck Software and North Bridge

        Lack of open source could cause...

        • Really expensive software
        • Control of big business
        • Reinvention of the wheel
        • Lack of community and collaboration
        • Limitation of technological development

          • Let's look at an enterprise and open source project...

          IBM Cloud and Carbon Components

          What is IBM Cloud?

        • Enterprise software
        • Not open source
        • Broken down into microservices

          • What is Carbon components?

        • Component library and design system
        • Vanilla JS, React components and style guidelines
        • All open source

          • How did they differ?

          IBM Cloud:

          • One person controlled all the code
          • No peer reviews to ensure security and accountability
          • Quality of code was bad

            • Carbon Components

            • Contributions from anyone
            • Peer reviews on all contributions
            • Testing
            • Good quality of code
            • Open for anyone to use

              • IBM Cloud code has improved because of the influence of Carbon Components üíØ

              It now has...

            • A peer review system
            • At least 85% testing coverage
            • Pair programming
            • Knowledge of the system among the whole team

              • Benefits OS provides:

              • Cost benefits
              • Community support
              • Quality of code
              • Transparency & reliability

                • offers flexibility and freedom

                ensures security and accountability

                How do you become part of this community and contribute to cool projects?

                I will specifically be showing how you can contribute using GitHub

                What is GitHub?

                • Code hosting platform for version control and collaboration
                • Uses Git
                • Users can pull down a repo and contribute back

                  • Finding a project to contribute to

                • Use GitHub Explore
                • Visit sites that have OS projects to contribute to (https://www.firsttimersonly.com/, https://www.codetriage.com/)
                • Follow peers to see what they're contributing to

                  • Your first contribution

                  • Find something easy (docs, small bugs)
                  • Look for specific labels: first timers only, good first issue, low hanging fruit, beginner, easy
                  • Work on something you're passionate about
                  • Things to consider: language, project, size
